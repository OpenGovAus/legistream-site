Info

Legistream is Open Gov Australia's solution to stream all nine Australian parliaments in one place.
It is designed to efficiently gather the various streams using open-source code, and to provide them to the Australian public in a simple way.

The back end

Legistream's background is where the real magic happens. The package scrapes parliament web pages and interacts with their APIs to obtain three key pieces of information: is the stream live, what is its URL, and which chamber of parliament is it associated with?

The front end

After this, the job is given to the front end (the site) to turn that raw data into something usable. The site uses video.js to play the stream URLs returned by legistream-backend on each parliament's page. The site runs on the Django web framework, and was built entirely from scratch.

Notes

Contribution is encouraged! If you have ideas to help improve Legistream, tell us, or make them yourself and open a pull request on GitHub.
